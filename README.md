
# **Don Macchiatos Website**

## **Project Overview**

This website was developed as a web development project focused on building a modern café website with both static and dynamic components. It showcases Don Macchiatos’ products, brand identity, and customer-facing features such as accounts and checkout.

## **Purpose**

To establish an online presence for Don Macchiatos, promote coffee products and seasonal offerings, and provide basic user interaction such as account access and order checkout.

## **Site Structure**

* `index.html` – Homepage / Coffee blog & promotions
* `products/index.html` – Product listings
* `about/index.html` – Company background and vision
* `locations/index.html` – Store locations and map
* `order/index.html` – Cart page (client-side)
* `order/checkout.html` – Checkout form and order processing
* `account/index.html` – Login and user dashboard
* `assets/` – Images, logos, icons, styles

## **Design Concept**
* **Color Scheme**: Warm coffee tones (brown, cream, white)
* **Typography**: Clean sans-serif with strong headers
* **Layout**: Grid-based product layout, centered content, responsive structure
* **UX Focus**: Simple navigation, readable content, minimal clutter
